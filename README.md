# ⌨️ Key Monitor Pro (macOS Safe)

A GUI-based keyboard event monitor built with Python and Tkinter.  
This version is fully compatible with macOS (M1/M2/M3) without requiring special permissions.

---

## 🚀 Features

- ✅ Key Press & Release detection
- ✅ Real-time logging
- ✅ Auto-save logs (TXT + JSON)
- ✅ Modern dark UI
- ✅ macOS safe (no crashes / no permissions)
